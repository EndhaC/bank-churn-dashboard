## Overview
This project analyzes customer churn data to identify why customers are leaving the bank. 
It was built using **Streamlit** as part of the Data Science Bootcamp assignment.

## Features
- **Interactive Filters:** Filter data by Gender and Card Type.
- **Visualizations:** Histograms and Scatter plots to explore churn drivers.
- **KPIs:** Real-time calculation of Churn Rate.

## How to Run
1. Clone this repository.
2. Run `pip install -r requirements.txt`.
3. Run `streamlit run streamlit_app.py`.
